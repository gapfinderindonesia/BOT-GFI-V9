## Clone this project

```bash
> git clone https://github.com/gapfinderindonesia/GFI-BOT-V8
```

```bash
>Command BOT /
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd GFI-BOT-V8
> bash install.sh
```

### Usage
```bash
> npm start
```
